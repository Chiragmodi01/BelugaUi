Buttons

Buttons are available in many types and customizable assets like shadow, shapes, sizes, hover effect, overlay, border, hollow etc.

Solid Buttons

To create a solid button use the base class btn, fill and add any of the following class to alter size - sm, md, lg, xl.

Colors of Buttons

You can customise your button's color, with your preferance. To choose different colors from primary, secondary, dark, danger, warning, success, info and lightgrey just add the reletive class to your button component.

Disabled Buttons

To create an disabled button use the base class btn-disabled.

Hover fill Buttons

To create an button with the hover effect which fills the color in button use the class hover-fill.

Link Buttons

To create an link button use the base class link and add any of the prefered border to the button. You can choose border from our utility class.

Shadow Buttons

To create an button component with shadow use the class shadow to the button.

Shadow on hover Buttons

To create an button component withe shadow on hover effect use the class hover-shadow.

Hollow Avatars

To create an hollow button component use the class hollow.

Hollow hover filled Buttons

To create an hollow button component which fills color on hover use the class hollow hover-fill.

Button with icons on left

To create an button component with icons on the left side use the class icon icon-left.

Button with icons on right

To create an button component with icons on the right side use the class icon icon-right.

Arrow hover effect Buttons

To create an button component with the arrow icon which shows cool transition on hover use the class transition-icon-btn icon icon-right.

Icon only Button

To create an icon only button component just add the icon at the place of button and use the classes mentioned below.

Icon only hover filled Buttons

To create an icon only button which fills the color on hover use the class hover-fill.

Simple icon only Buttons

To create an simple icon only button just add the icons at the place of buttons.

Full width Buttons

To create an full width button component use the class full-width .

Floating Buttons

To create an button component which floats on 4 different directions on your place and sticks there use the base class float-round-btn and add any of the following class to alter position - botton-r, bottom-l, top-r, top-l.

To create an outlined button component which floats on 4 different directions on your place and sticks there use the base class outline float-round-btn and add any of the following class to alter position - botton-r, bottom-l, top-r, top-l.
