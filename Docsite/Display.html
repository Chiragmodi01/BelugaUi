Display

We have some display properties as utility classes. So you can use them in your project without much hassle.

To use display: none; You can use the following class - no-display.
To use display: block; You can use the following class - block.
To use display: inline; You can use the following class - inline.
To use display: inline-block; You can use the following class - inline-block.
To use display: flex; You can use the following class - flex.
To use display: inline-flex; You can use the following class - inline-flex.
To use display: grid; You can use the following class - grid.
To use visibility: hidden; You can use the following class - hidden.
