Whole border

If you want to use the property border:none; You can just use the following class instead - no-border.

We have 4 different sizes of border - xs, sm, md, lg and they come with 2 different color which are - black and white. To use them you can just use the following class and change the size and color as per your preference - border-xs-black.

Border top radius

We have 6 different sizes of border top radius and also you can change the position from 'left' and 'right'. To use them you can just use the following class, if you want border-top-right-radius them use the following class - br-t-r-1. And if you want border-top-left-radius you can use - br-t-l-1.
If you want to apply radius to both of the sides for top, use the following classbr-t-1
You can change the number anywhere between 1 to 6 to alter radius as per your preference.
