Variables

We have some variables for many different styles properties like margin, border, padding, box-shadow, border-radius, gap, font-weights etc. So you can use them in your project without much hassle.

Font-weights

We have 4 types of font-weights, stored in variables --font-light --font-medium --font-semi-bold --font-bold.

Spacing

We have default values for for margin and padding stored in variables, just use --margin --padding and it'll give you default spacing on all sides.

Box-shadow

We have 2 types of default box-shadows stored in variables, light and dark --box-shadow-light --box-shadow-dark.

Border-radius & border

To use default value of border-radius and border stored in a variable, use --border-radius --border.

Flex gap

To use default value of flex-ap stored in a variable, use --flex-gap.

Flex-Direction

To use flex-direction:column; in flex. You can use the following class - flex-col.
To use flex-direction:row; in flex. You can use the following class - flex-row.
To use flex-direction:column-reverse; in flex. You can use the following class - flex-col-rw.
To use flex-direction:row-reverse; in flex. You can use the following class - flex-row-rw.

Flex-Direction

To use flex-wrap:wrap; in flex. You can use the following class - wrap.
To use flex-wrap:nowrap; in flex. You can use the following class - nowrap.
To use flex-wrap:wrap-reverse; in flex. You can use the following class - wrap-rw.
