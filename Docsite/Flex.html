Flex

We have some flex properties as utility classes. So you can use them in your project without much hassle.

Display

To use display: flex; You can just use the following class - flex.
To use display: inline-flex; You can just use the following class - inline-flex.

Centered

To use display: flex; align-items:center; justify-content:center; all together. You can just use the following class - flex-centered.

Align-items

To use align-items: center; You can just use the following class - align-center.
To use align-items: flex-start; You can just use the following class - align-start.
To use align-items: flex-end; You can just use the following class - align-end.
To use align-items: flex-base; You can just use the following class - align-baseline.

Justify-content

To use align-items: center; You can just use the following class - justify-center.
To use align-items: flex-start; You can just use the following class - justify-start.
To use align-items: flex-end; You can just use the following class - justify-end.
To use align-items: left; You can just use the following class - justify-left.
To use align-items: right; You can just use the following class - justify-right.
To use align-items: space-around; You can just use the following class - justify-space-around.
To use align-items: space-between; You can just use the following class - justify-space-between.
To use align-items: space-evenly; You can just use the following class - justify-space-even.

Gap

We have 8 different sizes of gaps in property gap in flex. You can use the following class and change the number anywhere between 1 to 8 to customize the gap size - gap-1.
We have 8 different sizes of column gaps in property column-gap in flex. You can use the following class and change the number anywhere between 1 to 8 to customize the gap size - col-gap-1.
We have 8 different sizes of row gaps in property row-gap in flex. You can use the following class and change the number anywhere between 1 to 8 to customize the gap size - row-gap-1.

Flex-Direction

To use flex-direction:column; in flex. You can use the following class - flex-col.
To use flex-direction:row; in flex. You can use the following class - flex-row.
To use flex-direction:column-reverse; in flex. You can use the following class - flex-col-rw.
To use flex-direction:row-reverse; in flex. You can use the following class - flex-row-rw.

Flex-Direction

To use flex-wrap:wrap; in flex. You can use the following class - wrap.
To use flex-wrap:nowrap; in flex. You can use the following class - nowrap.
To use flex-wrap:wrap-reverse; in flex. You can use the following class - wrap-rw.
