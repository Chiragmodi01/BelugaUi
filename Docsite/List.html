List

List component can be used at multiple places like showing items or notifications etc. We have lists in style of Ordered, Unordered, Roman and Stacked.

To use lists component, use the base class list-wrapper and addany list type of your choice to the parent wrapper. And give the class list-item to the individual list item.

Ordered List

Ordered list shows the list item with their index number, you can customize list style as per your preferance.

  1. Item One
  2. Item Two
  3. Item Three
  4. Item Four
