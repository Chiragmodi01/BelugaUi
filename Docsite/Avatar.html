Avatars

Avatar displays user image or initials as profile picture.
Avatars are available in many types and customizable assets like shadow, shapes, sizes, hover effect, overlay, border, types like image and text, multiples in group etc.

To create an avatar use the base class avatar and add the following class to alter size avatar-md. You can change the size as per your preferance, all the available sizes are xxs, xs, sm, md, lg, xl and xxl.

Round Avatar

To make the avatar round, use round class. That's it and you have a round user avatar.

Avatar with custom border-radius

You can customize the border-radius using our border-radius utility classes. To use it you can add the following class br-1 and change the number as per your preference from 1 to 6, where 6 is the maximum border-radius.

Avatar with border

You can customize the border using our border utility classes, in which you can change the size and color of the border. To use it you can add the following class border-xs-black and choose any of the following size class as per your preference xs, sm, md and lg. And the available color are black and white.

Avatar with shadow

Add the shadow using following class shadow-dark-sm. Shadow can be chossen between soft and dark and available sizes are sm, md, and lg.

Text Avatar

To create an text avatar, use the class avatar-text. And to customize the font-size, we have utility classes available. You can choose the font size from xs, sm, md, lg and xl, class should look like this f-sm.

Avatars in a group

To create a group of avatars, add the class avatar-group to the parent wrapper of all the avatars. And you can give each avatar some border, preferably same as the backgound color, to make it look good. The spacing between each avatar can also be customized by adding the class avatar-group-xl to the parent wrapper, in which you can change the size from all these - xxs, xs, sm, md, lg, xl and xxl.

Overlay effect on Avatars

To create an avatar with the overlay, add the following classes to parent wrapper avatar-container, img-hover-overlay. And add the following class to the overlay div inside the wrapper img-hover-overlay-item and overlay-round to use overlay for the round avatar. Add sizes as well for the overlay wrapper, according to the size of the wrapper. And add the class img-overlay-text to overlay text, inside overlay wrapper.

Custom Border-radius of overlay

To create an avatar with the overlay, add the following classes to parent wrapper avatar-container, img-hover-overlay. And add the following class to the overlay div inside the wrapper img-hover-overlay-item. You can change the border-radius of the overlay according to the avatar, using our border-radius utility classes. Add sizes as well for the overlay wrapper, according to the size of the wrapper. And add the class img-overlay-text to overlay text, inside overlay wrapper.

Avatar with hover effect

To add hover effect on avatar, add the class avatar-container and img-hover-overlay to the parent wrapper which contains the avatar. And it'll give you an smooth transition on hover.
