Alerts

Alerts displays notification or any type of warning.
Alerts are available in many types and customizable assets like shadow, shapes, sizes, hover effect, overlay, outline, hollow, border etc.

Solid Alerts

To create an solid alert component use the base class alert and any color of your preference from the following available ones primary, secondary, dark, liightgrey, success, warning, error and info.
