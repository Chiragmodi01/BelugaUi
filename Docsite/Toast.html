Toast

Toast are the components wich can be used to show any pop up notification to the user.
Toasts are available in many types and customizable assets like solid, border, color and dismiss etc.

Solid Toast

To create an solid toast use the base class toast and add any of the following class to alter it's color - basic, grey, lightgrey, success, warning, error, info.
