Heading textbox

These Solid textboxes are easy to integrate in your html and have a cool label-transition when you click on input field.
You can also customize your solid input. Add any following class to your input-container to alter textbox's size - sm, md, lg and we also have 8 different types of colors in textbox - primary, secondary, dark, lightgrey, success, warning, error and info. And the default one is in basic white color. To use these colors, Just add the class input and add any of the color mentioned above, following by an hifen(-) before.
Keep in mind that the for attribute of label element, should match to the id of input element. Both sould be unique to another set of input textbox, if you're using multiple.
